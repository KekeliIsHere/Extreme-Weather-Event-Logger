# Weather Event Logger

A simple Java application for logging and tracking extreme weather events with a graphical user interface.

### Steps to Run the application**
   - Run the MainWeather Class
   - The Weather Event Logger window will open

## Using the GUI

- **Add Events**: Use the input fields at the bottom to enter weather event details
- **View Events**: See all recorded events in the table at the top
- **Event Types**: Choose from Earthquake, Flood, Hurricane, or Wildfire
- **Risk Levels**: Assign risk levels to each event (Low, Medium, High, Critical)
- **Analysis**: View event analysis and statistics in the analysis panel

## Features
- Log different types of extreme weather events
- Track location and date of events
- Assign risk levels to events
- View all events in an organized table
- Analyze event data and patterns
